## 📖 简介

欢迎来到**寂静如雪**的个人博客。这里是我在数字世界里的一个小角落，主要用于：

* **技术沉淀**：记录自托管服务、服务器维护及各种技术方案的探索。
* **资源共享**：维护 24T 影视矩阵及各类实用软件工具。
* **生活记录**：在相册中分享壁纸收藏与摄影瞬间。

## ✨ 站点特色

* **极简主义**：采用纯色背景与网格布局，拒绝繁杂，专注阅读体验。
* **实用工具**：集成了 PDF 编辑、封面制作、镜像加速等多种在线服务。
* **极速访问**：基于 Astro 框架构建，享受丝滑的静态加载速度。
